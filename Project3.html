Unity VR Escape Room Project

Project Overview

This project showcases the development of innovative solutions and highlights its key outcomes.

Category: Programming
Language & Tools Used: Unity, C#
Year Published: 2024
Description: Escape the Iron Maiden is an immersive VR escape room game, created using Unity, where players must solve intricate puzzles to escape a medieval-themed iron maiden. With detailed surroundings and interactive elements, the game challenges players to use their problem-solving skills to progress through each level. The experience combines the thrill of an escape room with the power of VR, offering a unique and engaging adventure. Can you escape in time?
